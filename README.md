# DiscountBazaar.PK 🛍️

DiscountBazaar.PK is a decentralized, zero-inventory social commerce engine for Pakistan. It operates on a **Hybrid Storefront Model**, offering standard e-commerce retail alongside a viral "Squad Buying" mechanism. By integrating with local B2B dropshipping networks (HHC, YourMart) and utilizing SBP-compliant pre-authorization payment holds (Safepay), the platform delivers wholesale prices directly to consumers while remaining entirely asset-light.

## Features
* **Dual-Conversion Engine:** Standard "Buy Now" and "Join Squad" checkout flows on every product.
* **24-Hour Digital Escrow:** Secure 10% deposit holds via Safepay; 100% automated refunds if squads fail.
* **Dynamic Discounting:** Algorithmic price drops based on the number of members in a Squad.
* **WhatsApp OTP Auth:** Passwordless logins to eliminate fake COD orders.
* **Supplier & Admin Portals:** B2B dashboards for proposal generation, quality control, and logistics tracking.

## Technologies Used
* **Backend:** Node.js, Express, TypeScript
* **Database:** MongoDB, Mongoose
* **Web Portals (Buyer/Admin/Supplier):** Next.js, React, Tailwind CSS
* **Mobile App:** Flutter, Dart
* **Background Jobs:** BullMQ (Redis-backed)
* **Payments:** Safepay Node SDK
